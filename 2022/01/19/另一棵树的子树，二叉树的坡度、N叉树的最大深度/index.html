另一棵树的子树

TAG： #相同的树

题目

给你两棵二叉树root和subRoot。检验root中是否包含和subRoot具有相同结构和节点值的子树。如果存在，返回 true ；否则，返回 false 。

二叉树 tree 的一棵子树包括 tree 的某个节点和这个节点的所有后代节点。tree 也可以看做它自身的一棵子树。

思路

此处只需要去递归检查每棵子树和subRoot是否相同就行了，可以直接copy之前的相同的树的算法题。

解法

var isSameTree = function(tree1, tree2) {
  const checkTreeLeave = (tree1, tree2) => {
    if (!tree1 && !tree2) return true;

    if (tree1 && tree2) {
      if (tree1.val === tree2.val) {
        return checkTreeLeave(tree1.left, tree2.left)
        && checkTreeLeave(tree1.right, tree2.right);
      } 
      return false;
    } 
    return false;
  }
  return checkTreeLeave(tree1, tree2);
};

var isSubtree = function(root, subRoot) {
  let res = false;

  const checkIsSubTree = (root, subRoot) => {
    if (!root) return;

    if (!res) {
      res = isSameTree(root, subRoot);
      checkIsSubTree(root.left, subRoot);
      checkIsSubTree(root.right, subRoot);
    }
  }

  checkIsSubTree(root, subRoot);
 
  return res;
};

二叉树的坡度

TAG： #二叉树节点之和

题目

给你一个二叉树的根节点 root ，计算并返回 整个树 的坡度 。

一个树的 节点的坡度 定义即为，该节点左子树的节点之和和右子树节点之和的 差的绝对值 。如果没有左子树的话，左子树的节点之和为 0 ；没有右子树的话也是一样。空结点的坡度是 0 。

整个树 的坡度就是其所有节点的坡度之和。

思路

在外部设置一个res值，递归获取左右子树的坡度（坡度之和），然后去计算当前节点的坡度，并和res进行累加，最终返回这个res。

解法

/**
 * @param {TreeNode} root
 * @return {number}
 */
 var findTilt = function(root) {
  let res = 0;
  const computeTilt = (root) => {
    if (!root) return 0;
    const leftTilt = computeTilt(root.left);
    const rightTilt = computeTilt(root.right);
    const currentTilt = Math.abs(leftTilt - rightTilt);
    res += currentTilt;
    return root.val + leftTilt + rightTilt;
  }
  computeTilt(root);
  return res;
};

N叉树的深度

TAG： #递归 #每次递归取最深

题目

给定一个 N 叉树，找到其最大深度。

最大深度是指从根节点到最远叶子节点的最长路径上的节点总数。

N 叉树输入按层序遍历序列化表示，每组子节点由空值分隔（请参见示例）。

思路

N叉树的结构是一个数组，因此我们每遍历一层深度就要加1。我们在递归中写一个forEach循环来一层一层遍历它，每次遍历取当前当前深度和遍历的深度最大值，遍历完成后返回。

解法

/**
 * @return {number}
 * @param {
 *  val: number;
 *  children: { val:..., children: ... } [];
 * }
 */
 var maxDepth = function(root) {

  const getMaxDep = (root) => {
      if (!root) return 0;
      let maxDep = 0;
      root.children.forEach((child) => {
          maxDep = Math.max(maxDep, getMaxDep(child));
      });
      return maxDep + 1;
  }

  return getMaxDep(root);
};
