「ReactHooks」useReducer基础到进阶

Words count 4.3k Reading time 4 mins. Read count 0

  • React Hooks

前言

useReducer 是让无状态组件中运行的类似 redux 的功能 API 。

useReducer 基础

写法

useReducer 的写法为const [state, dispatch] = useReducer(reducer, initialArg, init)，第一个参数 reducer （必填），其写法和 redux 的 reducer 类似，是一个方法，通过 action.type 找到对应的 action 并执行。第二个参数 initialArg （必填），是 state 的初始值。第三个参数 init （非必填），当我们需要惰性初始化的时候就需要传入它，它的参数为上面提到的 initialArg，这么做可以将用于计算 state 的逻辑提取到 reducer 外部，这也为将来对重置 state 的 action 做处理提供了便利。useReducer 的返回值第一个是更新之后 state 的值，第二项是派发更新 state 的 dispatch 函数。

用法

  • 不需要惰性初始化
import { useReducer } from 'react'
function DemoUseReducer() {
    /* 传入reducer，初始化count为0 */
    const [ count, dispatch ] = useReducer((state,acton)=>{
        switch (acton.type) {
            case "add":
                return state + 1;
            case "sub":
                return state - 1;
            default:
                break;
        }
    },0)
    return (
        <>
            <h1>{ count }</h1>
            {/* 点击派发type为add的action */}
            <button onClick={ ()=>{ dispatch({ type: "add" }) } }>加</button>
            {/* 点击派发type为sub的action */}
            <button onClick={ ()=>{ dispatch({ type: "sub" }) } }>减</button>
        </>
    )
}

function App() {
    return (
        <DemoUseReducer />
    )
}
export default App
  • 需要惰性初始化
import { useReducer } from 'react'
/* 定义初始值 */
const initialCount = 0
/* 定义惰性初始函数 */
function init(initialCount) {
    return initialCount
}
/* 定义reducer */
function reducer(state, action) {
    switch (action.type) {
        /* action.type为increment，state（count）加1 */
        case 'increment':
            return state + 1;
        /* action.type为decrement，state（count）减1 */
        case 'decrement':
            return state - 1;
        /* action.type为reset，state（count）初始化为0 */
        case 'reset':
            return init(action.payload);
        default:
            throw new Error();
    }
}
function DemoUseReducer() {
    /* 传入reducer, initialCount, init */
    const [ count, dispatch ] = useReducer( reducer, initialCount, init )
    return (
        <>
            <h1>Count: { count }</h1>
            <button onClick={() => dispatch({type: 'reset', payload: initialCount})}>Reset</button>
            <button onClick={() => dispatch({type: 'decrement'})}>-</button>
            <button onClick={() => dispatch({type: 'increment'})}>+</button>
        </>
    )
}
function App() {
    return (
        <DemoUseReducer />
    )
}
export default App

useReducer 进阶

跳过 dispatch

原由和 useState 跳过 setState 一样，可以通过 useMemo 来优化。

「ReactHooks」useState 基础到进阶

模拟 Redux

React 不使用 state = initialState 这一由 Redux 推广开来的参数约定。有时候初始值依赖于 props，因此需要在调用 Hook 时指定。如果你特别喜欢上述的参数约定，可以通过调用 useReducer(reducer, undefined, reducer) 来模拟 Redux 的行为，但我们不鼓励你这么做。（官方建议，我还没碰见过这种场景🤡）
