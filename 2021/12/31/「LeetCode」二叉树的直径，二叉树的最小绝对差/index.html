二叉树的直径

#树 #最大深度 #递归

题目

给定一棵二叉树，你需要计算它的直径长度。一棵二叉树的直径长度是任意两个结点路径长度中的最大值。这条路径可能穿过也可能不穿过根结点。

思路

在外部定义一个变量 ans 始终存储最大的值；
然后开始递归，首先判空，为空返回 0，否则继续；
分别递归该节点的左右子树，返回其高度
将 ans 与左子树高度加右子树高度再加一的和(题干提到最终结果是取任意两个结点路径长度中的最大值，因此我们需要在脑中把节点为基点把整颗树平铺开来)取最大值
然后返回左右子树最大的深度并加 1。
如此一来我们最外层的 ans 变量会在每次递归确保自己是最长的路径。

代码

/**
 * @param {TreeNode} root
 * @return {number}
 */

var diameterOfBinaryTree = function (root) {
  let ans = 0;

  const getDiameter = (root) => {
    if (!root) return 0;

    const leftPath = getDiameter(root.left);
    const rightPath = getDiameter(root.right);

    ans = Math.max(ans, leftPath + rightPath + 1);

    return Math.max(leftPath, rightPath) + 1;
  };

  getDiameter(root);

  return ans - 1;
};

二叉树的最小绝对差

#树 #任意节点差的绝值对值 #递归

题目

给你一个二叉搜索树的根节点 root ，返回 树中任意两不同节点值之间的最小差值 。差值是一个正数，其数值等于两值之差的绝对值。

思路

由于需要找到任意两个节点的最小绝对差值，全员递归没得跑了；
这里采用一个 map 和一个 array(map 用来保存绝对差，array 用来保存节点的值)来实现；
先把树遍历一遍，将它的值寸与 array 中；
然后对 array 一个二层嵌套循环拿到所有的绝对差，并将其同时作为键和值放入 Map 中，放之前需要验一波重；
最后将 Map 序列化，取最小值。

代码

/**
 * @param {TreeNode} root
 * @return {number}
 */
var getMinimumDifference = function (root) {
  const hashMap = new Map();
  const all = [];

  const getAllNumber = (root) => {
    if (!root) return;
    getAllNumber(root.left);
    all.push(root?.val);
    getAllNumber(root.right);
  };

  getAllNumber(root);

  all.forEach((curItem, curIdx) => {
    all.forEach((item, idx) => {
      if (curIdx !== idx) {
        const res = Math.abs(curItem - item);
        if (!hashMap.get(res)) hashMap.set(res, res);
      }
    });
  });

  const mapToArray = Array.from(hashMap);

  return Math.min(...mapToArray.map(([val]) => val));
};
