树的众数

题目

给定一个有相同值的二叉搜索树（BST），找出 BST 中的所有众数（出现频率最高的元素）。

假定 BST 有如下定义：

  • 结点左子树中所含结点的值小于等于当前结点的值
  • 结点右子树中所含结点的值大于等于当前结点的值
  • 左子树和右子树都是二叉搜索树

例如：给定 BST [1,null,2,2]，返回[2]。

思路

建立一张 hash 表，将树节点的值作为 key 放入其中，value 作为该 key 出现的次数。从上至下遍历，每检测到一个 key 就对其加 1。

解法

首先 new 一张 Map，然后开始递归；

先判断节点是否为空，为空直接返回退出递归；不为空继续

判断表内是否已经存在这个 key 了，不存在添加，存在就加一；

操作完传入其左右节点向下遍历～

遍历结束后，将 Map 转化为数组，取出 value 最大的 key 即可。

代码

/**
 * @param {TreeNode} root
 * @return {number}
 */

var findMode = function (root) {
  const hashMap = new Map();

  const computeModeTimes = (root) => {
    if (!root) return;

    if (hashMap.get(root.val)) {
      hashMap.set(root.val, hashMap.get(root.val) + 1);
    } else {
      hashMap.set(root.val, 1);
    }

    computeModeTimes(root?.left);
    computeModeTimes(root?.right);
  };

  computeModeTimes(root);

  const mapToArray = Array.from(hashMap);

  const [, maxValue] = mapToArray.sort(
    ([, aValue], [, bValue]) => bValue - aValue
  )?.[0];

  return mapToArray
    .filter(([, value]) => value === maxValue)
    .map(([key]) => key);
};

二叉搜索树的最小绝对差

题目

给你一个二叉搜索树的根节点 root ，返回 树中任意两不同节点值之间的最小差值 。

差值是一个正数，其数值等于两值之差的绝对值。

思路

树中任意两不同节点值之间的最小差值，任意划重点，那枚举法没跑了。枚举所有的节点的差值然后取绝对值再取最小的。

解法

先取到所有的树节点的值，然后进行一次嵌套遍历取到所有的差值，最后取到最小值。

代码

/**
 * @param {TreeNode} root
 * @return {number}
 */
var getMinimumDifference = function (root) {
  const hashMap = new Map();
  const all = [];

  const getAllNumber = (root) => {
    if (!root) return;
    getAllNumber(root.left);
    all.push(root?.val);
    getAllNumber(root.right);
  };

  getAllNumber(root);

  all.forEach((curItem, curIdx) => {
    all.forEach((item, idx) => {
      if (curIdx !== idx) {
        const res = Math.abs(curItem - item);
        if (!hashMap.get(res)) hashMap.set(res, res);
      }
    });
  });

  const mapToArray = Array.from(hashMap);

  return Math.min(...mapToArray.map(([val]) => val));
};
