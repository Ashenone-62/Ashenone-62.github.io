树的最大深度

题目

给定一个二叉树，找出其最大深度。

二叉树的深度为根节点到最远叶子节点的最长路径上的节点数。

说明: 叶子节点是指没有子节点的节点。

思路

判断传入的节点是否为空，为空就返回0；不为空就去计算它左子树和右子树的高度，取最大的值返回。

这里取最大值，是因为如果已经递归到叶子节点了那势必会出现左右为0的局面，这时候累加一进入到父节点上，父节点可能也有其他子树或者没有子树，那么取两棵树的最大值肯定就是最大有效深度了。

/**
*				       (max(2,1)+1=3) 4
*			                   /         \
*        (max(1,1)+1=2) 2           5 (max(0,0)+1=1)
*       				       / \           
*      (max(0,0)+1=1) 1   3 (max(0,0)+1=1)
* */

解法

/**
 * @param {TreeNode} root
 * @return {number}
 */
var maxDepth = function(root) {
  const getDepth = (root) => {
    // 判断节点是否存在，不存在就返回0
    if (!root) return 0;
    // 计算左子树高度
    const leftHeight = getDepth(root.left);
    // 计算右子树高度
    const rightHeight = getDepth(root.right);
    // 取最大有效值加一后得到当前深度，并返回该值
    return Math.max(leftHeight, rightHeight) + 1;
  };

  return getDepth(root);
};

树的最小深度

题目

给定一个二叉树，找出其最小深度。

最小深度是从根节点到最近叶子节点的最短路径上的节点数量。

说明：叶子节点是指没有子节点的节点。

思路

查找树的最小深度要分为下面四种情况：

  1. 节点为空。
  2. 节点不为空，节点的左右子树同时为空。
  3. 节点不为空，节点的左右子树有一边为空。
  4. 节点不为空，节点的左右子树都不为空。

针对第一种情况，如果成立说明这棵树为空，返回0；

针对第二种情况，说明该节点为叶子节点，返回1；

针对第三种情况，返回左子树的高度+右子树的高度+1，左子树高度+右子树的高度因为有一边为空那么那边返回值肯定为0，因此直接相加不影响结果。

针对第四种情况，都不为空，我们取左子树和右子树的最小值+1，参考针对第三种情况的逻辑，第四种属于第三种的特殊情况。

解法

/**
 * @param {TreeNode} root
 * @return {number}
 */

var minDepth = function(root) {
  const getMinDepth = (root) => {
    // 节点为空
    if (!root) return 0;
    // 左右节点都为空
    if (!root.left && !root.right) return 1;
        // 获取左树高度
    const leftDepth = getMinDepth(root.left);
    // 获取右树高度
    const rightDepth = getMinDepth(root.right);
        // 左右节点有一边为空
    if (!root.left || !root.right) return leftDepth + rightDepth + 1;
        // 左右节点都不为空
    return Math.min(leftDepth, rightDepth) + 1;
  }

  return getMinDepth(root);
};

判断平衡二叉树

题目

给定一个二叉树，判断它是否是高度平衡的二叉树。

本题中，一棵高度平衡二叉树定义为：一个二叉树每个节点 的左右两个子树的高度差的绝对值不超过 1 。

思路

首先会有一个布尔值来标识这棵树是否平衡，如果它不平衡了就不会执行后续的判断逻辑了。

首先判断节点是否为空或者此树是否已经不平衡；

再然后获取左右子树的高度；

将他们相减取绝对值，如果结果大于了1说明已经不平衡，这时候将掌管平衡的变量置为false，退出函数即可。

如果绝对值没问题那就取左右子树的最大值+1返回，这里取最大值是树的深度泛指最大深度，除非特别说明是取最小或者其他深度。

解法

递归法：

/**
 * @param {TreeNode} root
 * @return {boolean}
 */
var isBalanced = function(root) {
  // 判断是否为空树
  if (!root) return true;
  // 定义平衡变量
  let balanced = true;
    // 
  const compareChildrenTreeDepth = (node) => {
    // 判断传入节点是否为空
    if (!node || !balanced) return 0; 
    // 获取左右子树高度
    const left = compareChildrenTreeDepth(node.left);
    const right = compareChildrenTreeDepth(node.right);
        // 根据左右高度绝对值来判断是否平衡
    if (Math.abs(left - right) > 1) {
      balanced = false;
      return 0;
    };
        // 取左右最大的值+1返回
    return Math.max(left, right) + 1;
  };

  compareChildrenTreeDepth(root);

  return balanced;
};

将有序数组转化为二叉搜索树

题目

给你一个整数数组 nums ，其中元素已经按 升序 排列，请你将其转换为一棵 高度平衡 二叉搜索树。

高度平衡 二叉树是一棵满足「每个节点的左右两个子树的高度差的绝对值不超过 1 」的二叉树。

思路

题目提到了升序，那么确定了我们的遍历方式为中序遍历。

我们每次遍历会传入原始数组、左和右的索引值，选择中间靠左的节点来做根节点；

如果左的值大于了右就说明到头了，返回null；

否则就用中心靠左的值创建一个节点；

然后再设置他的左树和右树，他们的左右索引值值都有中间向两边靠拢，左边是（left， mid - 1），右边（mid + 1， right）；

最终返回这个节点。

解法

/**
 * @param {number[]} nums
 * @return {TreeNode}
 */
var sortedArrayToBST = function(nums) {
      // 封装获取中间的值的函数
    const getMidIdx = (left, right) => Math.floor((left + right) / 2);
    const createTree = (nums, left, right) => {
          // 如果左边已经大于右边说明到头了
        if (left > right) return null;
                // 获取中间值
        const mid = getMidIdx(left, right);
          // 用中间值创建节点
        const node = new TreeNode(nums[mid]);
                // 在用mid和left/right的中间值去创建左树右树的节点
        node.left = createTree(nums, left, mid - 1);
        node.right = createTree(nums, mid + 1, right);
                // 最终返回这个节点
        return node;
    }

    return createTree(nums, 0, nums.length - 1);
};
