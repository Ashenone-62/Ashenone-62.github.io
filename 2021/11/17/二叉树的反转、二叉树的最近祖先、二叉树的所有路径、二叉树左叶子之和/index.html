二叉树的反转

题目

翻转一棵二叉树。

思路

思路很简单～

首先，判断一下节点是否为空，为空就返回null；

否则创建一个节点，将当前的节点的值赋给树节点的值，左右节点交换递归；（整个过程自上而下）

代码

/**
 * @param {TreeNode} root
 * @return {TreeNode}
 */

var invertTree = function(root) {
  const invert = (root) => {
    // 判空
    if (!root) return null;
    // 创建节点，第二三个参数是树节点，可以直接在这里搞递归套娃
    return new TreeNode(root?.val, invert(root?.right), invert(root?.left));
  }
  return invert(root);
};

二叉树的最近祖先

题目

给定一个二叉搜索树, 找到该树中两个指定节点的最近公共祖先。

百度百科中最近公共祖先的定义为：“对于有根树 T 的两个结点 p、q，最近公共祖先表示为一个结点 x，满足 x 是 p、q 的祖先且 x 的深度尽可能大（一个节点也可以是它自己的祖先）。”

思路

这个题给了一个很厉害的限制条件——传入的树肯定是二叉搜索树，秩序💪

值是有序的，那么当前节点的值小于了目标节点的值说明，目标节点肯定在当前节点的右边，我们只需要去右树寻找；同理，如果当前节点的值大于了目标节点，说明目标节点肯定在当前节点的左边，我们只用去左树即可。

有了上述的一个逻辑，我们就可以将对象的纬度降到数值的维度（第一版我想的使用判断两棵树是否相同来完成这道题🤡）。

首先初始化俩数组用于保存来目标节点的路径；

然后开始递归套娃，先判断节点是否为空，为空直接退出；不为空继续

然后判断是否为目标节点，是的话就推入对应的路径数组并退出；不是目标节点继续

然后判断当前节点的值和目标节点的值的大小，来确定去左子树还右子树寻找，最后推入当前节点的值到数组，表示已经到过这一路径了；

递归结束后，对两个目标节点的路径数组进行遍历，这里使用find只要找到就结束循环；由于我们将推路径放在递归的最后一步，因此路径是从下到上的，两个路径数组中索引值最小且相等的值即目标节点的最近祖先👌

代码

/**
 * @param {TreeNode} root
 * @param {TreeNode} p
 * @param {TreeNode} q
 * @return {TreeNode}
 */

var lowestCommonAncestor = function(tree, a, b) {
  // 初始化路径数组
  const pathA = [];
  const pathB = [];

  const getPath = (root, target, res) => {
    // 判空
    if (!root) return;
    // 判断当前节点是否为目标节点，是就将值推入路径数组，并退出递归；不是则继续
    if (root?.val === target) return res.push(root.val);
    // 判断当前节点和目标节点的大小关系，以此来确定寻找方向
    if (root.val < target) getPath(root.right, target, res);
    if (root.val > target) getPath(root.left, target, res);
    // 将当前节点的值推入路径数组
    res.push(root.val);
  }

  // 递归ing
  getPath(tree, a.val, pathA);
  getPath(tree, b.val, pathB);

  // 遍历得到结果，第一个想等的值即两个目标节点的最近祖先
  const res = pathA.find((nodeA) => {
    return pathB.find((nodeB) => {
      if (nodeA === nodeB) {
        return nodeA
      }
    })
  });
  // 用这个值构造一个树节点（题目要求的返回值是个TreeNode）
  return new TreeNode(res);
};

二叉树的所有路径

题目

给你一个二叉树的根节点 root ，按 任意顺序 ，返回所有从根节点到叶子节点的路径。

叶子节点 是指没有子节点的节点。

思路

首先初始化一个数组用来存放，每棵树的路径；

然后开始递归，首先万年不变的判空，节点为空直接返回当前的路径；不为空继续

然后使用模版字符串对当前节点的值进行字符串拼接，这里选择将箭头放在值的前面，然后通过判断path是否为空来确定当前拼接的节点是否是根节点；

再然后判断当前节点存不存在子树，如果已经是叶子节点了就返回当前拼接好的路径，并退出递归；

如果存在就继续递归，直至到达所有叶子节点。

代码

/**
 * @param {TreeNode} root
 * @return {string[]}
 */

var binaryTreePaths = function(root) {
  // 初始化路径数组
  const res = [];
  
  const getPath = (root, path = '') => {
    // 判空
    if (!root) return path;
        // 对路径进行累加，中间判断是否是根节点来决定箭头的存在
    path += `${path ? '->' : ''}${root?.val}`
        // 判断还有没有子树，如果没有就将当前路径推入路径数组，并退出递归
    if (!root?.left && !root?.right) return res?.push(path);
        // 否则就继续递归
    getPath(root?.left, path);
    getPath(root?.right, path);
  };

  getPath(root);

  return res;
};

二叉树的左叶子之和

题目

计算给定二叉树的所有左叶子之和。

思路

这里的也是采用递归的方式，且听我娓娓道来。

首先初始化一个数组用来装叶子的对象；

递归开始前，判断节点是否只是一个根节点，是直接返回结果0；不然就继续；

然后开始递归，开始也是万年不变的判空，为空就退出，不为空就继续；

再判断当前节点是否为非根节点的叶子节点，如果是，就将其值和一个标记是左叶子还是右叶子的布尔值一起包装成一个对象推入叶子数组中；（这里能直接推是因为根节点必不可能进入，所以能进入肯定是第二次或者再往后的递归了）

最后分为左右继续递归节点，是左标记值就传true，不是就传false。

递归结束后，通过数组的reduce方法，将左叶子节点捞出，并返回累加的值。

代码

/**
 * @param {TreeNode} root
 * @return {number}
 */

var sumOfLeftLeaves = function(root) {
  // 初始化左叶子数组
  const res = [];

  const getSumOfLeftLeaves = (root, isLeft = true) => {
    // 判空，为空直接退出，非空继续
    if (!root ) return;
        // 判断是否是叶子节点，是推入对象并退出；不是继续
    if (!root?.left && !root?.right) return res.push({
      val: root?.val,
      isLeft,
    });
        // 继续递归，分为左和右，左打上true的标记，右打上false标记
    getSumOfLeftLeaves(root?.left, isLeft = true);
    getSumOfLeftLeaves(root?.right, isLeft = false);
  }
    // 判断是否只存在根节点
  if (!Number.isNaN(Number(root?.val)) && !root?.left && !root?.right) return 0;
    // 递归
  getSumOfLeftLeaves(root);
    // 返回左叶子的累加值
  return res.filter((leaves) => leaves.isLeft).reduce((prev, cur) => {
    return prev += cur?.val;
  }, 0)
};
