二叉树的中序遍历

题目

给定一个二叉树的根节点 root ，返回它的 中序 遍历。

思路

通过递归去取节点的值，每次递归先判断这个节点是否为空，为空则不继续遍历。由于是中序遍历，所以遍历的顺序是左子树->根节点->右子树（这条规则是针对参与中序遍历的树的，例如根节点的左子树相当于是它自己的左子树的根节点）。

解法

递归法

/**
 * @param {TreeNode} root
 * @return {number[]}
 */
const inorderTraversal = function (root) {
  // 初始化结果数组
  const res = [];
  const inorder = (root) => {
    // 判断当前节点是否为空，为空则终止
    if (!root) return;
    // 先遍历左子树
    inorder(root.left);
    // 再取根节点
    res.push(root.val);
    // 最后遍历右子树
    inorder(root.right);
  };
  // 开始递归
  inorder(root);
  // 返回结果
  return res;
};

相同的树

题目

给你两棵二叉树的根节点 p 和 q ，编写一个函数来检验这两棵树是否相同。

如果两个树在结构上相同，并且节点具有相同的值，则认为它们是相同的。

思路

通过递归的方式来求解，每次递归开始判断一下传入的两个节点是否都为 null（如果两棵树从根节点一直顺利进行到叶子结点，那么它们的终点一定是 null），如果为真则直接返回 true，为假则返回 false。

接下来检验一下两个节点是否都存在，不是的话直接返回 false；结果为真则判断两数的这个节点的值是否相同，如果为真则分别将对应的左节点、右节点传入进行递归，为假就返回 false。

解法

递归法

/**
 * @param {TreeNode} p
 * @param {TreeNode} q
 * @return {boolean}
 */
var isSameTree = function (tree1, tree2) {
  const checkTreeLeave = (tree1, tree2) => {
    // 判断两个节点是否都不存在
    if (!tree1 && !tree2) return true;
    // 判断两个节点是否都存在
    if (tree1 && tree2) {
      // 判断两个节点值是否相同
      if (tree1.val === tree2.val) {
        // 对对应的左节点和右节点进行递归，左右都ok返回true
        return (
          checkTreeLeave(tree1.left, tree2.left) &&
          checkTreeLeave(tree1.right, tree2.right)
        );
      }
      return false;
    }
    return false;
  };
  return checkTreeLeave(tree1, tree2);
};

对称二叉树

题目

给定一个二叉树，检查它是否是镜像对称的。

思路

检查一棵树是否为镜像树，我们完全可以复用检查两棵树是否相同的算法思路。

他俩有啥区别呢？检测两棵树的是否相同的算法每次递归处理的是检测两棵树对应节点是否相同；而检测一棵树是否为镜像树每次递归则是检查一棵树的对称节点是否相同。

解法

递归法

/**
 * @param {TreeNode} root
 * @return {boolean}
 */
const check = (p, q) => {
  // 如果两个节点都不存在说明顺利的走到了叶子结点返回true
  if (!p && !q) return true;
  // 如果两个节点有一个不存在则说明不对称返回false
  if (!p || !q) return false;
  // 如果两个节点都存在，则值相等的情况下，左节点对应的右节点，右节点对应的左节点传入进行递归check
  return p.val === q.val && check(p.left, q.right) && check(p.right, q.left);
};
var isSymmetric = function (root) {
  return check(root, root);
};
