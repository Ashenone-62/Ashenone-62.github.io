「LeetCode」数的反转

Words count 2k Reading time 2 mins. Read count 0

  • Easy
  • LeetCode

题目

给你一个 32 位的有符号整数 x ，返回将 x 中的数字部分反转后的结果。

如果反转后整数超过 32 位的有符号整数的范围为大于 -2 的 31 次方，小于 2 的 31 次方减 1 ，如果不在范围内就返回 0。

假设环境不允许存储 64 位整数（有符号或无符号）。

思路

字符串法：将传入的 number 转化为 string，然后将 string 反转即可。

计算法：通过对传入的 number 取模，把 number 一位一位的推出，然后然后初始化一个值为 0，每次计算将这个值乘以 10 进位后在对推出的值进行累加，最终达到反转的结果。

解法

字符串

const max = Math.pow(2, 31) - 1;
const min = Math.pow(-2, 31);

const reverse_position = (num) => {
  // 转化为string，再将其序列化
  const numString = [...(num + "")];
  // 判断是否为负数，负数转化为字符串后第一位为符号
  if (Number.isNaN(Number(numString[0]))) {
    // 分离符号和数字
    const [symbol, ...nums] = numString;
    // 将数字反转后在和符号做拼接
    const reverseNum = symbol + nums.reverse().join("");
    // 判断反转后的数是否越过最小值后返回结果
    return reverseNum < min ? 0 : reverseNum;
  }
  // 不是负数就直接反转
  const reverseNum = numString.reverse().join("");
  // 判断反转后的数是否越过最大值后返回结果
  return reverseNum > max ? 0 : reverseNum;
};

计算

const max = Math.pow(2, 31) - 1;
const min = Math.pow(-2, 31);

const reverse_compute = (num) => {
  // 将形参用个变量保存一下
  let originValue = num;
  // 初始化结果
  let result = 0;

  // 当原始值不为0的情况下就一直循环
  while (originValue !== 0) {
    // 对原始值取模，拿到推出的数
    const push = originValue % 10;
    // 原始值再除以10进行退位
    originValue = parseInt(originValue / 10, 10);
    // 结果先乘10进行进位再加上推出的值
    result = result * 10 + push;
    // 判断每次计算后的结果是否在最大最小值范围内
    if (result > max || result < min) {
      result = 0;
    }
  }
  // 返回结果
  return result;
};
