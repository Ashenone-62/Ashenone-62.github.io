「LeetCode」扁平化嵌套列表迭代器

Words count 1.9k Reading time 2 mins. Read count 0

  • LeetCode
  • Normal

给你一个嵌套的整型列表。请你设计一个迭代器，使其能够遍历这个整型列表中的所有整数。列表中的每一项或者为一个整数，或者是另一个列表。其中列表的元素也可能是整数或是其他列表。

我的思路

这道题思路是利用深度优先（DFS）来解，但 JS 迭代器那块的了解不够，没有解出来，现在已搞懂。官答的两种方案的一种也是利用 DFS 来解，还有一种是利用栈来解决。

踩坑

本题定义了一个 NestedInteger ，它可以存储 number 类型或 Array，类似于一棵多叉树，每个节点都可以有很多子节点。后面我们需要编写的 next 与 hasNext 相当于重载原型上的方法，因为我们做的就是一个迭代器嘛。这道题还提供了下面 3 个方法（已经封装好了）：

  • isInteger() ，判断当前存储的对象是否为 number。
  • getInteger() , 如果当前存储的元素是 number 型的，那么返回当前的结果 number，否则调用会失败。
  • getList() ，如果当前存储的元素是 Array 型的，那么返回该 Array，否则调用会失败。

官答

var NestedIterator = function (nestedList) {
  //定义一个数组，放目标值
  vals = [];
  //定义一个函数将原始数组传入
  const dfs = (nestedList) => {
    //迭代原始数组
    for (const nest of nestedList) {
      //判断一下其中的元素是不是number类型
      if (nest.isInteger()) {
        //是，我们就把他放入目标数组
        vals.push(nest.getInteger());
      } else {
        //不是，我们就调用getList()将返回的结果传入DFS执行
        dfs(nest.getList());
      }
    }
  };
  //执行DFS
  dfs(nestedList);
};
//重载hasNext
//如果目标函数的长度大于0，就返回true，表明接下来还有元素
NestedIterator.prototype.hasNext = function () {
  return vals.length > 0;
};
//重载next
NestedIterator.prototype.next = function () {
  //当发生下次迭代的时候，迭代器会移到第一个和第二个元素之间
  const val = vals[0];
  //除去经过的元素（也就是每次迭代的第一个）
  vals = vals.slice(1);
  //返回经过的元素（也就是每次迭代的第一个）
  return val;
};

总结

写完上面的内容，感觉 DFS 并不是很好理解，以栈、指针的形式来处理会更好，也更符合迭代器的理念。栈处理去此处
