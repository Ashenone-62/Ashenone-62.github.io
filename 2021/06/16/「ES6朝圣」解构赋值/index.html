前言

解构赋值绝对是 ES6 带来最好用以及现在最常用的功能之一！

数组解构

只要数据的数据解构存在 Iterator 接口，它就能被解构。

基本用法

const [one, two, three] = [1, 2, 3];

console.log(one, two, three);

// 1, 2, 3

真相只有一个

const [ badGuy, goodGuy ] = [ '张三'，'好人'，'好人'，'好人' ];

console.log(badGuy, goodGuy);

// badGuy -> 张三
// goodGuy -> ['好人'，'好人'，'好人' ]

不完全解构

const [test1] = [];
const [test2, test3] = [1];

console.log(test1, test2, test3);

// undefined，1，undefined

默认值

解构赋值允许设置默认值

const [one, two = 2] = [1];

console.log(one, two);

// 1, 2

成员严格等于 undefined 默认值才生效

const [member = 2] = [null];

console.log(member);

// null

默认值可以为方法

const func = () => {
  console.log("Hello World");
};

const [test1 = func()] = [1];
const [test2 = func()] = [];

console.log(test1);
// 1
console.log(test2);
// Hello World
// undefined

默认值可以用其他解构获得的值

const [x = 1, y = x] = []; // x=1，y=1
const [x = 1, y = x] = [2]; // x=2，y=2
const [x = 1, y = x] = [1, 2]; // x=1，y=2
const [x = y, y = x] = []; // y还没被声明，报错

对象解构

基本使用（只能通过属性名来解构，或者通过赋值的形式来重新命名）

⚠ 这里重命名是先找到同名属性，然后再赋给对应的变量

⚠ 前者是模式，后者是变量

const abc = {
  a: "aaa",
  b: "bbb",
  c: "ccc",
};

const { a, b, c } = abc;
const { a: A, b: B, c: C } = abc;

console.log(a, b, c);
// aaa, bbb, ccc

console.log(A, B, C);
// aaa, bbb, ccc

解构方法

const { log } = console;

log("Hello World");

坑

用已经声明的变量来解构赋值

let x;
{ x } = { name: '张三' };
// 报错，js把{ x }识别成了一个代码块

({ x } = { name: '张三' });
// 正常执行

对象的方式解构数组

const { 0: one, 1: two, 2: three } = [1, 2, 3];

console.log(one, two, three);

字符串解构

基本使用

const [a, b, c, d, e] = "Hello";

console.log(a, b, c, d, e);

// h e l l o

解构长度

⚠ 了解下包装类型

const { length: len } = "张三";

console.log(len);

// 2

数值与布尔值解构

对象方式解构 number 类型和 boolean 类型会将其转换成对象

⚠ 了解下包装类型

const { toString: numberToString } = 1;
console.log(numberToString === Number.prototype.toString);

// true

const { toString: booleanToString } = true;
console.log(booleanToString === Boolean.prototype.toString);

// true

行车不规范,亲人两行泪

数组解构的数据是 number，false，NaN，undefined，null，{}，则报错

// 全员报错
const [test1] = 1; // {}解构可行，转换成对象
const [test2] = false; // {}解构可行，转换成对象
const [test3] = NaN;
const [test4] = undefined;
const [test5] = null;
const [test6] = {};

不要轻易使用圆括号，关爱 JS 引擎，人人有责
